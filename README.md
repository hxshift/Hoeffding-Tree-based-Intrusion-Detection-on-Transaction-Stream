# Hoeffding-Tree-based-Intrusion-Detection-on-Transaction-Stream
Real-time intrusion detection system using the Hoeffding Tree algorithm to detect anomalous transactions in continuous data streams. Supports incremental learning, low-latency predictions, and adapts to evolving patterns without retraining. Its suitable for dynamic learning environments. The project uses Docker based Kafka for streaming deployment, and River for online machine learning. Detection results are automatically saved to a CSV file for further analysis.
